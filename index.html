Tentang Saya

Sejak pertama kali belajar membuat halaman sederhana dengans HTML dan mempercantiknya dengan CSS, saya langsung tertarik dengan dunia coding. Rasanya seru ketika kode yang saya tulis bisa berubah menjadi tampilan web yang nyata dan bisa digunakan dan saya juga sedang belajar Javascript. Saat ini, saya fokus memperkuat dasar-dasar pengembangan web, terutama membuat tampilan yang rapi, menarik, dan enak dilihat.

Di luar kegiatan belajar coding, saya suka bermain Futsal dan bermain Sepak Bola. Saya percaya aktivitas dan kreativitas di luar coding bisa bantu saya dapat ide baru untuk membuat halaman web yang lebih bagus.
